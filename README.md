## Intro

This a simple accounting app powered by asp.net core.

## Docker Suppport

### Build docker image

``` bash
docker build -t weihanli/accountingapp .
```

### Run in container

``` bash
docker run -P --rm --name accountingapp-demo weihanli/accountingapp
```
